加粗问题

如果两个加粗字体中间有未加粗的符号。未加粗符号应与 ** 相隔至少两个空格。

效果：

隔一个空格：加粗字体 →↑↓←😀 加粗字体

隔两个空格：加粗字体 →↑↓←😀 加粗字体

如果中间是文字并且不空格，

加粗字体未加粗字体加粗字体

修改首页副标题时报错

在 _config.fluid.yml 文件（主题配置文件）中，通过修改 index.slogan.text 来修改副标题。

index:
  slogan:  # 首页副标题的独立设置
    enable: true  # 为 false 则不显示任何内容
    text: 'An elegant Material-Design theme for Hexo'  # 为空则按 hexo config.subtitle 显示

由于，副标题的内容被单引号 ' 括起，因此当副标题内容含有单引号时，副标题的配置无法成功执行。

解决方案：通过将单引号 ' 换为双引号 " 即可。

index:
  slogan:  # 首页副标题的独立设置
    enable: true  # 为 false 则不显示任何内容
    text: "Welcom to Guoguo's blog!"

Tag 插件-便签

便签的写法有两种，一种是 {% %} 形式，一种是 HTML 形式

但是 HTML 形式不能在多行的情况下使用

代码：

{% note success %}
第一行
第二行
第三行
{% endnote %}

效果：

第一行
第二行
第三行

代码：

<p class="note note-primary">
第一行
第二行
第三行
</p>

效果：

第一行 第二行 第三行
