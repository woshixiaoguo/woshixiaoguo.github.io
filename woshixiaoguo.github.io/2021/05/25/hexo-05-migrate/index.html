博客迁移

思路

  1. 创建 Hexo 文件夹

  2. 克隆博客仓库

  3. 替换配置文件

由于 hexo init 必须执行于 空文件夹，因此无法在博客仓库中初始化 Hexo 。

创建 Hexo 文件夹

hexo init [destination]

删除配置文件

删除 Hexo 文件夹下的 source 文件夹和 .gitignore、_config.landscape.yml、_config.yml 三个文件。

设置仓库

cd blog

git remote add origin 仓库地址

git pull origin master:master

下载主题文件

将主题文件拷贝至 themes 文件夹中并改名为 fluid

导入静态资源

  1. 将 woshixiaoguo.github.io\img 中的图片拷贝至 博客路径/themes/fluid/source/img

  2. 修改 博客路径/themes/fluid/languages 目录下的 zh-CN.yml 文件

安装部署插件

npm install hexo-deployer-git --save
